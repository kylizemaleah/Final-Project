# Final-Project
PUBH 422 Final Project - NHIS Dataset

Project Overview
This final project is designed to assess your ability to manage, analyze, and visualize public health data using R Studio. You will act as a public health analyst, starting from raw data and culminating in a professional report. The project integrates all key skills from the course: data sourcing, cleaning, recoding, descriptive and inferential statistics, visualization, and scientific communication.
